Shutterfake

Shutterfake is an image gallery website. The name Shutterfake comes from the inspiration of this project - Shutterstock.

The website is static (made with HTML and CSS) and doesn't contain any Javascript. The displayed images are randomly generated from Unsplash and get updated every time the page is refreshed.

The website contains a minimal layout and page count. It is composed of 4 pages:

  • Images page
  • Image detail page
  • Story page
  • About page
